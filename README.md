# 4d-plugin-gd-v3
Process images using GD Graphics Library.

## Build Information

build manually so that freetype, fontconfig, raqm, imagequant are disabled.

## Library version

* `libgd-2.3.3`
* `libavif-0.11.1` - `libaom-3.6.1`
* `libpng16`
* `libjpeg`
* `libtiff`
* `libwebp`

## Rendering options

|-|-|
